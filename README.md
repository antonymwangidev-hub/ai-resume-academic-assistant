# AI-Powered Resume & Academic Writing Assistant (Ethical)

A web-based AI application that helps job seekers and students generate:
- Personalized resumes (PDF download)
- Tailored cover letters
- Ethical academic writing improvements with citation guidance

## Key Features
- ATS-friendly resume generation
- Downloadable PDF resumes
- Prompt chaining & role matching
- Ethical AI safeguards
- Academic integrity enforcement

## Tech Stack
- Python
- Streamlit
- OpenAI API
- ReportLab (PDF generation)

## Ethical Notice
This tool assists with writing and formatting only.
Users are responsible for reviewing, editing, and ethically submitting all content.
